# Flowers Page

A simple beginner HTML page displaying two flower images that link out to their Wikipedia pages.

## Files

| File | Description |
|------|-------------|
| `flowers.html` | Single HTML page with inline styles |
| `_.jpg` | Image of a Tulip |
| `_ (1).jpg` | Image of a Lotus (Nelumbo nucifera) |

## HTML Concepts Used

- `<h1>` heading with inline `style` for `text-align: center`
- `<a href>` anchor tags wrapping `<img>` elements for clickable image links
- `target` attribute not set (links open in the same tab)
- `<img>` with `width` attribute set to `200px` for fixed display size
- `src` attribute pointing to local image files
- Wikipedia URLs as `href` values for the two flowers
